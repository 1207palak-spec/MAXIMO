# MAXIMO

MAXIMO is an interactive 3D web experience built for an Electrical Engineering & Power Infrastructure company.

Instead of using a traditional landing page, the project presents the company's business through a continuous cinematic journey. As users scroll, they travel through different parts of the electrical ecosystem including power generation, transmission, substations, industrial automation, smart grids, and renewable energy.

The goal of this project is to combine modern web technologies with real-world engineering concepts to create an engaging and visually rich experience.

---

## Features

- Interactive 3D scenes built with React Three Fiber
- Smooth camera transitions driven by scroll
- Realistic electrical infrastructure environments
- Modular scene architecture
- GSAP-based animations
- Responsive UI overlay
- Optimized rendering for modern browsers

## Getting Started

Clone the repository

```bash
git clone https://github.com/1207palak-spec/MAXIMO.git
```

Move into the project directory

```bash
cd MAXIMO
```

Install dependencies

```bash
npm install
```

Start the development server

```bash
npm run dev
```

Create a production build

```bash
npm run build
```

Preview the production build

```bash
npm run preview
```

## Design Goal

The objective is to build a premium engineering experience that represents how electrical power moves from generation to end users.

The experience focuses on:

- Power Generation
- High Voltage Transmission
- Substations
- Industrial Automation
- Smart Grid
- Renewable Energy
- Electrical Infrastructure

rather than following a traditional multi-section website layout.
